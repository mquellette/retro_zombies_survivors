# Proof of Concept — План

## Стек
- **Telegram Mini App** (WebApp API)
- **HTML5 Canvas** + **JavaScript** (чистый, без движка для PoC)
- Portrait: 360×640 логических пикселей

## Этапы PoC

### Этап 1 — Скелет UI (без ассетов)
- [ ] Главный экран: кнопка «Играть» (простой div/canvas)
- [ ] Экран выбора уровней: 1 уровень «Кладбище»
- [ ] Переход в игровую сцену

### Этап 2 — Базовый геймплей
- [ ] Игровое поле (тайловая карта, можно заглушки цветом)
- [ ] Персонаж: движение виртуальным джойстиком
- [ ] Авто-атака (ближайший враг, таймер)
- [ ] Враги: спавн волнами, движение к игроку
- [ ] Коллизии (враг↔игрок, снаряд↔враг)
- [ ] HP игрока, HP врагов, смерть врагов
- [ ] XP-гемы (выпадают из врагов, подбор)
- [ ] Таймер раунда (2-3 минуты для теста)

### Этап 3 — Базовый HUD
- [ ] HP-бар
- [ ] Таймер
- [ ] XP-бар / уровень
- [ ] Счётчик убийств

### Этап 4 — Game loop
- [ ] Level up → выбор улучшения (3 варианта)
- [ ] Game Over экран со статистикой
- [ ] Кнопка «Заново»

### Этап 5 — Замена заглушек на ассеты
- [ ] Подключение спрайтов игрока
- [ ] Подключение спрайтов врагов
- [ ] Тайлы окружения
- [ ] UI из Figma
- [ ] Эффекты

## Геймплей-параметры (стартовые, для балансировки)

| Параметр | Значение |
|---|---|
| Скорость игрока | 2 тайла/сек |
| HP игрока | 100 |
| Урон базовой атаки | 10 |
| Скорость атаки | 1 выстрел/сек |
| Скорость врага (зомби) | 1 тайл/сек |
| HP зомби | 20 |
| Урон зомби | 5 (при касании, кулдаун 1 сек) |
| XP за зомби | 1 |
| XP до level up | 5, 10, 15, 20... (+5 каждый уровень) |
| Длительность раунда | 180 сек |
| Спавн врагов | каждые 2 сек, +1 враг каждые 30 сек |
